# Flappy Bird Game

A browser-based Flappy Bird game built with Next.js and TypeScript.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the game.

## Testing

Run unit tests:

```bash
npm test
```

Run tests in watch mode:

```bash
npm run test:watch
```

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `lib/` - Game logic modules (physics, collision, rendering, etc.)
- `components/` - React components (Game component)
